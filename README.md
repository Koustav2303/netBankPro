NetBankPro - Modern Banking Dashboard
A premium, highly interactive frontend banking dashboard built to showcase modern fintech UI/UX patterns.

NetBankPro is a strictly frontend application featuring seamless page transitions, global state management for real-time balance syncing, and immersive micro-interactions without requiring a backend.

Key Features
Theme Toggle: Seamless dark and light mode integration across the entire application.

Global State Management: Real-time balance updates across all components using React Context API.

Premium Animations: Smooth page routing, 3D flipping ATM cards, and interactive sliders powered by GSAP.

Authentication Flow: Clean login and signup pages with toast notifications instead of intrusive native browser popups.

Advanced Routing: Multi-page layout using React Router v6 with a persistent sidebar and mobile-friendly glassmorphic top navigation.

Dashboard Modules
Account Overview: Real-time balance display, income/expense statistics, and quick action widgets.

Card Management: Interactive 3D debit card with flip animation, master freeze toggle, and dynamic spending limit sliders.

Fund Transfers: Deposit and withdraw interface featuring a simulated OTP security verification modal and session history.

UPI Payments: Camera scanner UI simulation, recent contacts carousel, and shareable QR code generation.

Fixed Deposits: Interactive investment calculator with visual projection bars and a digital certificate generator.

Transaction History: Dual-view (list/grid) statement ledger with searchable data, category icons, and detailed digital e-receipts.

Contact & Support: Dual-column layout with premium contact cards and a simulated messaging form.

Tech Stack
React 18 (scaffolded with Vite)

Tailwind CSS version 3

GSAP (GreenSock Animation Platform)

React Router DOM

React Hot Toast

Lucide React (for iconography)

Getting Started
Prerequisites
Make sure you have Node.js installed on your local machine.

Installation Steps
Clone the repository
git clone https://github.com/Koustav2303/netbankpro.git

Navigate to the project directory
cd netbankpro

Install all dependencies
npm install

Start the development server
npm run dev

Deployment
This project is fully optimized for deployment on platforms like Vercel or Netlify. Simply connect your GitHub repository to your preferred platform, and it will automatically detect the Vite build settings.
